# Computer Vision Projects

A collection of computer vision projects.

## Projects

### [Air Canvas](air_canvas/) — Draw in the air using your webcam

![Air Canvas screenshot](air_canvas/screenshot.png)

Uses a pretrained hand-tracking model (MediaPipe) to follow your index
finger through your webcam and turns its movement into drawing on
screen — no mouse, no touchscreen. Pick colors and an eraser by holding
up two fingers and hovering over an on-screen toolbar.

See [air_canvas/README.md](air_canvas/README.md) for setup and usage
instructions.
